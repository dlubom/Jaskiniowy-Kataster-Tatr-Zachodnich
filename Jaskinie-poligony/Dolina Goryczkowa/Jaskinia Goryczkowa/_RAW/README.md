# Jaskinia Goryczkowa — surowe materiały źródłowe

- **Źródło**: Krzysztof Dudziński
- **Autorzy pomiarów**: Peter Imrich, Tina Majerníčková i inni
- **Data pomiarów**: od 2012-03-10 (wg dat w plikach Therion)
- **Dodał**: Dariusz Lubomski
- **Kompletność**: pełny projekt Therion z pomiarami, rysunkami i eksportami

## Pliki

- `Goričkova.zip` — archiwum ZIP z projektem Therion
- `Goriczkova/` — rozpakowana zawartość:
  - `index.th` — główny plik projektu Therion
  - `thconfig` — konfiguracja kompilacji Therion
  - `data/` — pliki pomiarowe (g1-g5.th), rysunki (gorickova1-4.th2), zdjęcia (gor1-3.JPG), dane źródłowe (.txt)
  - `output/` — wyeksportowane pliki: PDF (plan, przekrój, wersje PL/EN), KML, LOX, XVI
